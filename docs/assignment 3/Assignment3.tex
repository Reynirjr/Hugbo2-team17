\section{Project Status}

\subsection{Implemented}

The following user stories and features have been implemented:

\begin{itemize}
    \item \textbf{US1 (View menu):} Restaurant list and menu browsing with prices and availability.
    \item \textbf{US5 (Basket management):} Add items to basket and view basket contents.
    \item \textbf{US3 \& US4 (Place order, Confirm order):} Item details screen and full checkout flow.
    \item \textbf{US2 (View order status):} Order status screen showing order state.
    \item \textbf{US6 (Login with phone number):} Phone number entry and session management.
    \item \textbf{US7 (Vegetarian filter):} Filter menu to show only vegetarian items.
    \item \textbf{US8 (Estimated wait time):} Estimated wait time shown before ordering.
    \item Backend integration with Supabase; \texttt{BasketService} and \texttt{SessionManager} in place.
\end{itemize}

We have completed the core ordering flow (menu, basket, checkout, order status), phone login, vegetarian filter, and estimated wait time.
